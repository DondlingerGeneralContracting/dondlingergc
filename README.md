# Dondlinger General Contracting

Official website for Dondlinger General Contracting.

## GitHub Pages

This website is deployed using GitHub Pages. The site is automatically deployed when changes are pushed to the `main` branch.

### Local Development

To view the website locally, simply open `index.html` in your web browser.

### Website Structure

- `index.html` - Main website page
- `styles.css` - Styling and responsive design
- `.github/workflows/pages.yml` - GitHub Pages deployment workflow

## Features

- Responsive design for mobile, tablet, and desktop
- Professional company presentation
- Service showcase
- Contact information section
